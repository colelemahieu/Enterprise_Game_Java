# Enterprise Java Game

## Linux Instructions

### Clone the repository
```bash
git clone https://github.com/colelemahieu/Enterprise_Game_Java.git
cd Enterprise_Game_Java
```

### Run the launch game
```bash
./run_game.sh
```

## Requirements
- **Java Development Kit (JDK) 11 or higher**
- **Linux/macOS/Windows** with GUI support
- **X11 or XWayland** (for Linux users on Wayland)
